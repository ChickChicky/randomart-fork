## Quick Start

### Generate file

```console
$ cc -o nob nob.c
$ ./nob run file output.png
$ feh output.png
```

### Compile to GLSL shader and display interactively

```console
$ cc -o nob nob.c
$ ./nob run gui
$ feh output.png
```
